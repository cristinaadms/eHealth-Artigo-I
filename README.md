# eHealth-Artigo-I

Este repositório contém a cadeia de caracteres de busca, os dados, scripts e resultados relacionados à análise de literatura científica na área de eHealth, com ênfase em dois recortes temáticos: contexto geral de eHealth e monitoramento de pacientes.

## Estrutura do Repositório

### Pastas

**arquivos**  
Contém os artefatos associados à etapa de busca e seleção dos estudos, incluindo:
- Os seis filtros aplicados no processo de revisão;
- A cadeia de caracteres (string de busca) utilizada nas bases de dados.

**data**  
Armazena as bases de dados finais resultantes do processo de filtragem:
- `filtro04.csv`: conjunto de dados referente ao contexto geral de eHealth;
- `filtro06.csv`: conjunto de dados referente ao monitoramento de pacientes.

**processamento**  
Contém os notebooks responsáveis pelo pré-processamento dos dados:
- Notebook associado ao Filtro 04;
- Notebook associado ao Filtro 06.

**resultados_filtro04**  
Contém os resultados das análises referentes ao Filtro 04.

**resultados_filtro06**  
Contém os resultados das análises referentes ao Filtro 06.
